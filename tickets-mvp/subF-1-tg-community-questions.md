## Мета
Зняти два бізнес-критичні невідомі до написання коду.

## Задачі
- [ ] Знайти офіційну TG-групу mono API (лінк на сторінці api.monobank.ua/docs)
- [ ] Питання 1: чи прийнятний сценарій, коли користувачі самостійно вставляють персональні токени у сторонній сервіс (PFM/трекер підписок) на етапі MVP до апруву provider API?

- [ ] Зафіксувати відповіді у нотатках проєкту

## Acceptance Criteria
Обидві відповіді отримані та задокументовані; рішення по юр. формі (фізособа/ФОП/ТОВ/поки нічого) прийняте.
